# Studio Pottery Website

A contemporary, editorial-style Next.js website for a British studio pottery brand.

## Features

- **Editorial Design**: Large serif headlines, clean typography, generous spacing
- **Gallery-Style Layout**: Vertical, non-grid product listings
- **Warm Palette**: Earthy colors (clay, paper, celadon, moss)
- **Minimal UI**: Clean, confident, modern aesthetic

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- **Homepage** (`/`): Full-screen hero with large headline
- **Collection** (`/collection`): Vertical gallery-style product listing
- **Product** (`/collection/[id]`): Editorial product page with gallery label style

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Google Fonts (Fraunces for headlines, Inter for body)

## Design System

### Colors
- `clay`: #C97D60
- `paper`: #F5F1E8 (background)
- `celadon`: #9FB8AD
- `moss`: #6B7F5A
- `earth`: #8B6F47
- `stone`: #D4C4B0

### Typography
- **Headlines**: Fraunces (serif)
- **Body**: Inter (sans-serif)
